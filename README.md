# Usage
  ### Create the environment
  ```
  conda create -n reasoning_shortcuts python=3.9
  ```
  ### Activate it
  ```
  conda activate reasoning_shortcuts 
  ```
  ### Install requirements
  ```
  pip install -r requirements.tx
  ```
